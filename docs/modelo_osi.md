# Modelo OSI

El modelo OSI esta compuesto por **7 capas**, cada una con una funcion especifica.

---
![Modelo OSI](img/osi.png)
---

## Capas del modelo OSI

| Nº | Capa                 | Función principal |
|----|-----------------------|------------------|
| 7  | Aplicación           | Interaccion con el usuario |
| 6  | Presentacion         | Formatos y cifrado |
| 5  | Sesion               | Control de conexiones |
| 4  | Transporte           | Segmentacion y control |
| 3  | Red                  | Enrutamiento |
| 2  | Enlace de datos      | Control de acceso al medio |
| 1  | Física               | Transmision de bits |

---

## Ejemplo práctico


Navegar por una web:
**Capa 7** – HTTP
**Capa 4** – TCP
**Capa 3** – IP
**Capa 1** – Señal electrica por el cable
