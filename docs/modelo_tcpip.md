# Modelo TCP/IP


El modelo TCP/IP es el que realmente se utiliza en Internet.

---
![Modelo TCP/IP](img/tcpip.png)
---

##  Capas del modelo TCP/IP

| Modelo TCP/IP (4 capas) | Equivalencia en OSI |
|--------------------------|---------------------|
| Aplicación               | 5, 6, 7 |
| Transporte               | 4 |
| Internet                 | 3 |
| Acceso al medio          | 1, 2 |

---

## Protocolos habituales

- **HTTP, DNS, SMTP** (Aplicación)
- **TCP, UDP** (Transporte)
- **IPv4, IPv6** (Internet)
- **Ethernet, WiFi** (Acceso al medio)

---

## Ejemplo de uso


Descargar un archivo:
**TCP** gestiona la conexión
**IP** se encarga del direccionamiento
**Ethernet** transmite las tramas
